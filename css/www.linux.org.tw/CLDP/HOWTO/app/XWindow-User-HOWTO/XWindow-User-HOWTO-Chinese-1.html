1. 說明

X 視窗系統是由 Massachusetts Institute of Technology 所 開發出來的，並於 1984 年首次開放使用。它是個先進的、網路透明化、視窗化及 圖形環境的系統。本文假設你己經成功地安裝了 X ，而且它已經正常工作了。所以 我們所討論的是如何善用 X ，而不是如何去編譯 X 。另外也是因為大多數 的套件，在安裝時，只要選用 X ，就己能自動安裝 X 了。

1.1 其它資料來源

如果你才剛入門 XFree86 HOWTO 及 XFree86 Video Timings HOWTO 兩書是更好的選擇。在找得到他們的地方，常常也可以找到本文件。本文的後面 還有參考書目，也不耍忘了讀一下那些 首頁。

1.2 版本

若有更新版本，會不定期在 comp.os.linux.help 上 刊登，也會上載到相關 Linux WWW 及 FTP 網站 (含 Linux Documentation Project 網站)。 一般而言，最新版常常是放在 http://www.croftj.net/~ray/howto 上， 其中原來是發行 Matelab 寶庫。

在 1.1 版中，增加了對於 rxvt 的說明，擴增對 X 資源的說明，增加了一些有關 X 的 歷史，更新其中指到 SunSite 的 MetaLab 的連結，也說明了如何在 X 中使 用 Type 1 字型的指令。

在 1.2 版中，增加了許多在 X 中使用 TrueType 字型的說明，一些設定 xdm 的 細節，一些新增的資源。同時也更正了我對空白螢幕特性的討論(謝謝 Heinrich Langos 的更正)。

在 1.3 版中，包含一些更正，有關一些新版套件中字型路徑的錯誤，以及許多有關 X 的基本保全問題。我也增加了一些有關 KDE 及 E-Mail 的訣竅。

在 1.4 版中，新增了一些來自 Anthony J. 的更正及補充，還有一些由 Tomasz Motylewski 的有關保全的訣竅。

在 2.0 版中，包含一些由 Guus Bosch 、 Brian J. Miller 及我自己的更正，還有 許多新的更新及資訊，同時希望有人能繼續維護此文件。
