11. 參考資料及其餘資料來源

有關 X 視窗，在你所用的電腦上、在網際網路上、在你喜歡的書店出的一些優良書藉中 ，有許多可以找得到的資訊。不妨試試下述的來源，它們對筆者而言，有極多助益，說不定對 你而言，也有極多助益。

  • 看一下 Linux Documentation Project 網站，其中有許多關於 Linux 、 X 及其它相關的文件。
  • http://www.x11.org/有點像是所有有關 X 的交換中心。
  • http://www.themes.org/是有關各種視窗管理員的佈景主題的總部。
  • X 公會的網站是 http://www.x.org/... ，但好像搬 到 http://www.opengroup.com/ 去了。
  • http://www.xfree86.org/中有 XFree86 的資料。
  • O'Reilly 系列有關 X 視窗的書嗎！到 http://www.ora.com/ 中找一下正確書名。
  • 有關在 X 中使用 TrueType 的更多資訊， 在 http://www.freetype.org/ 中。
  • X 、 xterm 、 XFree86 或其它常用的用戶程式 的參考頁，都很有用而且對於資訊都相當簡約，所以也很建議一讀。就如一個老生常 談 - RTFM。
  • 有個 Remote X Apps MINI-HOWTO ，它對於了解如何在 X 上跑近端 及遠端用戶程式，頗有助益。
  • 不要忘了到 http://www.gnome.org/ 及 http://www.kde.org/ 看看最近 Linux 桌面環境被整合成什麼 樣子，自古以來這方式己經變得越來越像一個真理了。
